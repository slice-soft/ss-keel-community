## Reporting a Vulnerability

**Do not open a public GitHub issue for security vulnerabilities.**

If you discover a security issue in any SliceSoft project, please report it privately by emailing:

**slicesoft@juancadev.com**

Include in your report:

- Affected project and version
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix if you have one

We will acknowledge your report within **48 hours** and aim to release a fix within **7 days** depending on severity.

We appreciate responsible disclosure and will credit you in the release notes unless you prefer to remain anonymous.
